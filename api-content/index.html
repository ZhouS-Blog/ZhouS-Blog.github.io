{"posts":[{"title":"使用Gridea搭建静态博客网站","content":"示例网站：zp-blog.github.io 1.注册一个GitHub账户 https://github.com/join 2.创建一个GitHub项目，点击Start a Project 然后我们创建一个仓库，用于存放代码（放心，不用你写代码，Gridea会把代码存进来），xxx是你的GitHub用户名，Repository name为用户名.github.io，设置为public，下面的不用写 3.设置KEY 我们点击右上角的头像，点击Settings 找到Developer Settings，点进去，找到Personal Access Token，点击Generate new Token Note设置为用户名，勾选repo，提交后复制密钥，推出了就要重新申请了。 4.配置Gridea 下载 https://github.com/getgridea/gridea 并安装 打开gridea，找到远程，平台选择GitHub Pages，域名和仓库名称填入用户名.github.io，分值为master，仓库用户名填写GitHub用户名，令牌则为刚刚获取到的KEY。 我们点击保存，再点击检测远程连接，如果成功，则点击同步，然后打开浏览器，输入xxx(用户名).github.io，就可以访问自己的网站了！ 下一篇文章将会为大家带来更多Gridea技巧！ ","link":"https://zp-blog.github.io/gridea-01/"}]}
